# Fair Credit Assessment: Evaluating Synthetic Data for Credit Scoring

An empirical study evaluating whether CTGAN-generated synthetic data can replace real consumer credit data in credit scoring models вЂ” comparing real vs synthetic across three dimensions: predictive performance, fairness, and explainability.

**MSc Big Data Analytics Dissertation вЂ” Sheffield Hallam University, 2025**

---

## Research Question

Can synthetic data generated by CTGAN serve as a viable substitute for real consumer credit data in training credit scoring models, without sacrificing predictive accuracy, fairness, or interpretability?

---

## Datasets

This project uses two publicly available datasets. Due to licensing restrictions, data files are not included in this repository.

| Dataset | Source | Records | Complexity |
|---|---|---|---|
| Give Me Some Credit (GMSC) | [Kaggle](https://www.kaggle.com/c/GiveMeSomeCredit) | 150,000 | Lower |
| Home Credit Default Risk (HC) | [Kaggle](https://www.kaggle.com/c/home-credit-default-risk) | 307,511 | Higher |

Download each dataset from Kaggle and place the CSV files in the data/ folder before running any notebooks.

## Methods

### Models
- **XGBoost** вЂ” gradient boosting, high predictive performance
- **Logistic Regression** вЂ” linear baseline, maximum interpretability
- **Explainable Boosting Machine (EBM)** вЂ” interpretable by design

### Synthetic Data Generation
CTGAN was used to generate synthetic versions of both datasets.

### Evaluation Dimensions
- **Predictive performance** вЂ” AUC-ROC, F1, precision, recall
- **Fairness** вЂ” demographic parity and equal opportunity by age group
- **Explainability** вЂ” SHAP feature importance comparison
